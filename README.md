# Saba Vafakhah — Portfolio

Personal portfolio site for Saba Vafakhah — UX Researcher & Product Designer.
Static HTML / CSS / JS, no build step. Bilingual (EN / DE).

## Run locally

```bash
python3 -m http.server 4321
```

Then open <http://localhost:4321>.

## Structure

- `index.html` — markup (hero, work case studies, about, skills, experience, contact)
- `css/styles.css` — all styles (navy design system, animations)
- `js/app.js` — intro, custom cursor, language toggle, reveals, interactions
- `assets/` — portrait + case-study screenshots
